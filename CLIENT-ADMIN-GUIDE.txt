Rama Bakery Admin Guide
=======================

Admin link:
yourdomain.com/#admin

Default login:
Username: admin
Password: Rama@123

How to upload/change photos:
1. Login to admin.
2. The first section is "Start Here: Upload / Change Photos".
3. For logo, hero, menu or contact image, click "Upload / Replace".
4. For gallery or fresh bake photos, click "Add Photos".
5. Choose JPG, JPEG or PNG image from laptop/mobile.
6. Click "Save Changes" or "Save Image Changes".
7. Click "View Website" and refresh.

Important:
- Upload alone stores the photo, but Save Changes publishes it on the website.
- If image upload fails, open yourdomain.com/setup-check.php.
- uploads_folder_writable and data_folder_writable must be true.
- On GoDaddy File Manager, set assets/uploads and data permissions to 755 or 775 if needed.

Before giving to client:
Change the admin password in api/config.php.

AUTO LOGOUT SECURITY
- Admin panel now logs out automatically after 15 minutes of no activity.
- Activity means mouse movement, click, typing, touch, or scroll.
- In the last 60 seconds, it shows a warning countdown.
- To change the time, open api/config.php and edit ADMIN_IDLE_TIMEOUT_SECONDS.
  Example: 10 minutes = 10 * 60
